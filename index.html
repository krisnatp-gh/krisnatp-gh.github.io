Works

Olist Seller Churn Modeling

Collaborative work between me and my colleagues. Olist is a marketplace platform, which generates revenue through seller commission. I developed a classification model to forecast seller churn on a quarterly basis and prepared the churn dataset from the transaction data. Using Hyperopt optimization, the model achieved of 92% recall after tuning.

Hyperopt Demo: Insurance Claims Prediction

Made educational content demonstrating Hyperopt library implementation for efficient hyperparameter tuning. Applied this approach to a travel insurance claims prediction case with severe class imbalance, showcasing advantages over Grid and Random search methods. Achieved significant performance improvement, doubling test recall from 41% to 82% with minimum iteration.

Predictive Regression Analysis for Apartment Pricing

Developed a regression-based Automated Valuation Model to predict apartment prices in Daegu, South Korea. The model analyzes property characteristics to generate accurate price estimates for property owners and real estate agents, achieving 82% accuracy in price predictions.

Exploratory Data Analysis of Transjakarta

Analyzed ridership data across 100+ Transjakarta routes using April 2023 transaction records. Examined route utilization, travel duration, and demographic patterns to identify critical congestion points and underutilized routes, providing actionable insights to improve public transportation service efficiency.
